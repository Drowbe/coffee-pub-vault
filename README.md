## Description

Optional assets (sounds, images, and related content) for use with the Coffee Pub suite of modules, integrated via **Coffee Pub Blacksmith**.

## Requirements

- **FoundryVTT**: Version 13.x
- **Coffee Pub Blacksmith**: Required dependency — provides shared services and functionality

## Installation

1. Inside Foundry VTT, use the following manifest URL:
   ```
   https://github.com/Drowbe/coffee-pub-vault/releases/latest/download/module.json
   ```
2. Enable the module in your game world's module settings
3. Ensure Coffee Pub Blacksmith is installed and enabled

## Usage

Enable the module after Blacksmith. Additional usage notes will be documented as assets and features are added.

## Configuration

Module settings can be found in: Configure Settings → Coffee Pub Vault
